# 진우퀀트 — 구동원리 해설 (한 사이클이 실제로 어떻게 도나)

> 위 `진우퀀트_파라미터_해설.md`의 **실제 값**이 각 단계에서 무슨 값으로 어떻게 쓰이는지 순서대로. 초보 기준, 용어 괄호 풀이. production 무수정(설명만).

## 본체 v3.7.2 — 월 1회 사이클 (실제 흐름)

### 1단계 · 데이터 수집
- `fetch_price_panel()`이 **KOSPI 18종 + KOSPI지수**의 일별 종가를 모은다(FDR).
- *왜 일별*: Mom12(12개월)·Echo(먼 과거 6개월)·β(60일) 같은 가격 기반 점수를 계산하려면 과거 시계열이 필요.

### 2단계 · 팩터 점수 계산 (종목마다)
각 종목에 대해 실제 산식이 이렇게 돈다:
- **체력_12점** = F_korean(0~9) × 12/9.001 → 예: F_korean 9면 12점(재무 만점).
- **Mom12**: 최근 12개월 수익률을 보고 `≥+60%→+2`, `≥+30%→+1`, `−10%~+30%→0`, … → **추세 강하면 가점**.
- **BAB**: 60일 베타가 `<0.4→+2`(방어주), `<0.7→+1`, `≥1.5→−2`(고변동) → **덜 출렁이면 가점**.
- **Echo**: 1년 전~7개월 전 수익률을 18종끼리 줄세워 상위 20% `+1`, 하위 20% `−1`(×1.0).
- **FAR**: 한 달새 −5%↓인데 체력 9↑면 `+2`(과대낙폭 우량주 역발상).
- 여기에 외부입력 ModF·Sloan·NOA(분기 갱신)를 더한다.
- → **체력_최종 = 이 모두의 합.** (숫자가 클수록 "재무 튼튼 + 추세 좋음 + 덜 위험".)

### 3단계 · 등급·비중
- 체력_최종을 등급컷에 통과: **S+ ≥14 / S ≥12 / A ≥9** → 이 셋만 **매수 대상(picks)**.
- picks를 **1/n 동일가중** → **종목 15% 컷** → **섹터 35% 컷** → 합 100% **정규화** = 권장비중.
- *예 의미*: picks가 10종이면 1/10=10%씩 시작, 어느 것도 15% 못 넘고, 반도체 합이 35% 넘으면 그 섹터만 비례 축소.

### 4단계 · (병행 관찰) PEAD 가산 — 운용 미반영
- 실적공시 후 **60거래일 이내** 종목만 SUE(깜짝실적 표준화) 3분위 ±1 → **×0.5**를 체력_최종에 얹어 `체력_v39`.
- 지금은 **기록만**(운용은 v3.7.2). 9월 판정에서 채택 여부 결정.

### 5단계 · MA200 추세 체크 (행동규칙)
- 월 1회, 시장(또는 보유)이 **10개월 이동평균 위/아래**인지 눈으로 점검. 아래면 RISK-OFF(관망).
- *주의*: 이걸 자동 디리스킹으로 정교화하면 검증서 손해(8회 기각). 그래서 **단순 점검만**.

### 6단계 · 산출·집행
- `v37_2_scores_latest.csv` + 대시보드 생성. 진우가 **VWAP(거래량가중평균가) 이하 지정가**로 직접 매수. 비용 **왕복 0.235%** 반영.

## 테마 lane — forward 사이클 (재량 + 가드레일)

### ① 스캔
- `kosdaq_monthly_scan` + 3계층 엔진을 돌려 후보를 surfacing(떠올리기). **매수신호 아님**.

### ② 3계층 점등 (2개 이상이어야 후보)
- **A 촉매**: 뉴스(종목명+수주·공급계약…) / DART 공시.
- **B 확산**: `美연료 ON + 한국대장 +10%↑ + peer가 대장보다 15%p 뒤처짐 + peer 급등(+100%) 아님` → "따라잡기 후보".
- **C 수급**: 외국인·기관 순매수. (A공시+C수급 2+ 점등=주목)

### ③ thesis·무효화 등록 (사기 전에)
- 워치리스트에 thesis 1줄 + 무효화 트리거 + 선반영 점검을 **먼저** 적는다(사후 합리화 차단).

### ④ 사이징 (2%룰 + ATR)
- `risk_trade = 0.02 × lane자본`, `stop = 2.5 × ATR(14)`, `shares = risk_trade / stop`.
- 워치는 일ATR ≈7.3%로 커서 **수량이 자동으로 작아짐**. 손절폭 −20% 넘으면 진입 스킵.

### ⑤ 집행·한도
- 분할 진입(정찰 1/3). 동일 테마 ≤2종·lane40%, 1종 ≤lane25%, 보유 ≤5.
- 킬스위치: lane **일−7%/주−12%/월−20%** 닿으면 신규 정지+thesis 재검토.

### ⑥ 측정 (Track W)
- 매월 (테마픽 − 같은돈 시스템픽) 기록. **6개월 누적 ≥0이면 직관이 밥값** → 유지, 음수 지속 → 축소.

## "이 숫자가 크면/작으면" 빠른 사전

| 숫자 | 크면 | 작으면/낮으면 |
|---|---|---|
| 체력_최종 | 상위 등급(S+≥14) → 매수 대상 | F(<0) → 미편입 |
| Mom12 점수 | +2(강한 상승추세) | −2(하락추세, 회피) |
| β(베타) | 고변동 → BAB −2 | 저변동 방어주 → BAB +2 |
| Echo | 먼과거 강세 +1(반복 경향) | 먼과거 약세 −1 |
| SUE | 깜짝실적 강함 → +0.5 가산 | 실적 실망 → −0.5 |
| 일ATR% | 변동 큼 → 수량 작게(2%룰) | 변동 작음 → 수량 크게 |
| 리드-래그 갭 | peer 따라잡기 여지 큼(후보) | 갭 작음 → 이미 반영(제외) |
| MA200 이격 | 추세 위(보유) | 아래(RISK-OFF 관망) |
| Track W 누적 | +면 직관 기여(유지) | −면 직관 손해(축소) |

> 한 줄: **점수가 높은 18종 중 S+/S/A만, 고른 비중에 상한을 씌워 사고, 추세·비용·방어는 규칙이 자동 처리** — 그리고 테마는 미리 적은 thesis로만 직관을 쓰고 전부 사후 측정한다.
