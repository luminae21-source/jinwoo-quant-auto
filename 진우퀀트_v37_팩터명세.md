## 1. 12M Momentum (Mom12)

### 학술 백본
- **Jegadeesh & Titman (1993)** — "Returns to Buying Winners and Selling Losers"
- 한국 시장 검증: 김민기 (2014), 한국증권학회지

### 정의
최근 12개월 수익률에서 직전 1개월을 제외한 11개월 수익률 ("11-month, skip 1-month").
- **왜 1개월 skip:** 단기 reversal (FAR가 잡는 신호)과 분리하기 위해.
- v3.6 FAR이 이미 1M reversal을 다루므로 Mom12는 **순수 추세** 측정.

### 계산식
```python
r_mom12 = price[t-21] / price[t-252] - 1
# 영업일 기준: 21일 ≈ 1개월, 252일 ≈ 12개월
```

### 데이터 소스
- FDR (`FinanceDataReader`), 종목코드 12개월 일별 종가
- v3.6 `get_1m_returns()` 함수 확장하여 12M도 동시 수집

### ±2 정수 변환 룰

| r_mom12 | Mom12 점수 | 의미 |
|---|---|---|
| ≥ +60% | **+2** | 강한 상승 추세 (LIG넥스원 케이스) |
| +30% ~ +60% | +1 | 상승 추세 |
| -10% ~ +30% | 0 | 중립 |
| -30% ~ -10% | -1 | 하락 추세 |
| < -30% | **-2** | 강한 하락 추세 (모멘텀 sell) |

**임계값 근거:** 한국 시장 4년 평균 누적 +20~30% 수준 (KOSPI). +60%는 약 2σ.

### v3.6 충돌 검토
- F·ModF·Sloan: 펀더멘털 (분기 손익/현금흐름) — 시간 척도 다름. **충돌 없음**.
- FAR: 1M reversal — Mom12는 12M trend. **시간 척도 분리**.
- 다중공선성 우려: 1M과 12M 수익률 상관 ~0.1~0.2 (한국). **낮음**.

### v3.6 universe 예상 영향
| 종목 | 예상 r_mom12 | Mom12 | 결과 |
|---|---|---|---|
| LIG넥스원 | +50%~+100% (방산 사이클) | +1 ~ +2 | F=-0.16 약점 보강 |
| NAVER | -5% ~ +10% | 0 | 디커플링 케이스 확인 |
| 알테오젠 | +20% ~ +50% | 0 ~ +1 | 추가 가점 |
| 카카오 | -20% ~ -5% | 0 ~ -1 | 약점 부각 |

## 2. BAB (Betting Against Beta)

### 학술 백본
- **Frazzini & Pedersen (2014)** — Journal of Financial Economics
- 한국 시장: Mizuno (2020) 일본·한국 패널 — β slope 유의
- 핵심: 저β 종목이 risk-adjusted 기준 초과수익. 고β는 leverage constraint 가진 투자자가 과매수 → underperform.

### 정의
60거래일 rolling β (vs KOSPI 종합지수).

### 계산식
```python
β_60d = Cov(r_종목, r_KOSPI) / Var(r_KOSPI)
# 일별 수익률 60일 윈도우
```

### 데이터 소스
- FDR로 종목 + KOSPI 지수 (`KS11`) 60일 일별 종가
- v3.6 인프라 확장으로 데이터 수집 비용 미미

### ±2 정수 변환 룰

| β_60d | BAB 점수 | 의미 |
|---|---|---|
| < 0.7 | **+2** | 저β — risk-adjusted 우위 |
| 0.7 ~ 0.9 | +1 | 다소 저β |
| 0.9 ~ 1.1 | 0 | 시장과 동조 |
| 1.1 ~ 1.3 | -1 | 다소 고β |
| > 1.3 | **-2** | 고β — leverage 부담 |

**임계값 근거:** Frazzini-Pedersen 글로벌 평균 β = 1.0, 표준편차 ~0.3. ±1σ 기준.

### v3.6 충돌 검토
- F·ModF·Sloan·NOA: 펀더멘털 — β는 가격변동성. **차원 다름**.
- FAR: 1M 가격 + 체력 — BAB는 60일 β. **방향 다름**.
- Mom12: 12M trend — BAB는 변동성 자체. **상관 ~0.1**.

### v3.6 universe 예상 영향
| 종목 | 예상 β_60d | BAB | 결과 |
|---|---|---|---|
| KT&G | 0.5~0.7 | +1 ~ +2 | 방어주 우대 |
| KB금융·삼성물산 | 0.7~0.9 | +1 | 안정 우대 |
| 삼성전자·SK하이닉스 | 1.0~1.2 | 0 ~ -1 | 시장 동조 |
| 알테오젠·카카오 | 1.3~1.8 | -1 ~ -2 | 고변동 페널티 |
| 두산에너빌리티·LIG넥스원 | 1.2~1.5 | -1 ~ -2 | 사이클성 페널티 |

**주의:** BAB는 Mom12·FAR과 결합 시 "고β + 강모멘텀" 종목(LIG넥스원)에서 점수가 상쇄될 수 있음. 이게 의도된 효과 — risk-adjusted 우대.

## 3. NOA (Net Operating Assets)

### 학술 백본
- **Hirshleifer, Hou, Teoh, Zhang (2004)** — Journal of Accounting and Economics
- "Do Investors Overvalue Firms With Bloated Balance Sheets?"
- 한국 시장: 김상규·박영석 (2011) — 누적 영업이익 / 자산 비율 유의

### 정의
```
NOA_ratio = (Operating Assets - Operating Liabilities) / Lagged Total Assets

영업자산  = 총자산 - 현금 및 현금성자산
영업부채  = 총부채 - 차입금 - 우선주
```

### 단순화 (한국 DART 기준)
```
NOA_ratio = (총자산 - 현금성자산 - 총부채 + 차입금) / 직전기 총자산
```

### 데이터 소스
- DART API (분기보고서 재무상태표)
- 필요 항목: 총자산, 현금성자산, 총부채, 차입금 (장기·단기 합산)
- **분기 1회 수동 입력** (`JINWOO_v37` dict에 ModF·Sloan과 동일 방식)
- 또는 향후 DART API 호출 자동화 (`step4_dart.py` 패턴 활용 가능)

### ±2 정수 변환 룰 (산업 중립화)

산업 내 z-score 변환 후:

| NOA z-score | NOA 점수 | 의미 |
|---|---|---|
| < -1.0 | **+2** | 자산 매우 효율적 |
| -1.0 ~ -0.5 | +1 | 효율적 |
| -0.5 ~ +0.5 | 0 | 중립 |
| +0.5 ~ +1.0 | -1 | 자산 비효율 |
| > +1.0 | **-2** | 자산 매우 비효율 (overvalued) |

**산업 중립화 이유:** NOA 절대값은 산업별로 크게 다름 (반도체 vs 금융 vs 화장품). 산업 내 비교가 의미 있음.

**18종목 universe의 산업 분포가 11개 산업에 펼쳐져 있어 산업 내 N=1~2인 경우 많음.** → 초기 버전은 z-score 대신 **산업 중앙값 대비 편차** 또는 **단순 절대 임계값** 사용 가능. 백테스트 결과로 결정.

### v3.6 충돌 검토
- F_korean: ROA·자산회전율 등 일부 포함 → NOA와 약한 상관 (~0.2~0.3)
- Sloan: 현금흐름 측 accrual → NOA는 재무상태표 측. **상관 ~0.3, 보완적**.
- ModF: 산업 보정 F — NOA는 직접 산업 z-score. 보강 관계.
- **결론:** 약한 상관 있으나 정보 차원 다름. F·Sloan과 함께 회계 품질 3축 완성.

### v3.6 universe 예상 영향 (가설)
| 종목 | 가설 NOA | 이유 |
|---|---|---|
| KT&G | 매우 효율적 (NOA -1 ~ -2 → +2) | 자산경량, 현금부유 |
| 삼성전자 | 효율적 (NOA -0.5 → +1) | 반도체 내 1위 효율 |
| 삼성SDI | 비효율 (NOA +1 → -2) | 배터리 capex 누적 |
| 두산에너빌리티 | 비효율 (NOA +0.5~+1 → -1 ~ -2) | 원전 플랜트 자산 |
| 삼성물산 | 중립 (NOA ~0 → 0) | 다각화 |

**실측은 DART 분기 데이터로 확인 필요** — 영역 1의 후속 작업.
