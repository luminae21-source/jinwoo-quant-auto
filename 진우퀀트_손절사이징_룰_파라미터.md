# 1. 본체 — 포트폴리오 변동성 타겟팅 (❌ 검증 결과 기각 — 2026-06-07 갱신)

> **검증 완료**: `validate_voltarget.py` 실측 결과 **모든 변형이 base Sharpe(2.35)를 못 넘음** → 기각. 상세: `검증결과_voltarget_atr_2026-06-07.md`. 아래 원안은 *기록용으로* 남기되, **MDD를 −9%대로 누르는 게 목적이고 CAGR 대폭 희생을 수용할 때만 18% EWMA** 변형을 고려. 그 외 도입 안 함.

**근거 수치.** 실현 연변동성 25.6%, 월 σ 7.40%.

**룰 (제안 seed).** 목표 연변동성 **20%**(월 환산 5.77%)로 설정하고 총노출을 조정:
```
σ_hat_t   = 최근 3개월 포트폴리오 월수익률 표준편차 (또는 EWMA, λ=0.84)
exposure_t = min(1.0, 0.0577 / σ_hat_t)     # 레버리지 금지 → 상한 1.0
cash_t     = 1 − exposure_t                  # 나머지는 현금
```
- 목표 20% 선택 이유: 현행 25.6%에서 과하게 내리면(예: 12%) Sharpe 2.96의 상방을 너무 깎음. **20%는 약 −22% 노출 축소**에 해당, 워스트 구간(−9.79% 달) 직후 자동 디레버리징.
- 검증 포인트: 목표를 {16/18/20/22%}, 윈도우 {2/3/6개월}로 그리드 → net Sharpe·MDD·회전율 트레이드오프 확인. **상한 1.0 고정(차입 금지).**

**한계.** σ 추정 후행 → 급락 직후 축소가 1개월 늦을 수 있음. 월간 시스템이라 반응 지연 불가피.

# 2. 테마 lane — 2% 룰 + ATR 연동 사이징 (지금 적용)

이 lane은 **종목별 하드 손절이 전제**이므로 손절거리로 수량이 자동 결정된다.

**핵심 공식 (워치리스트 `진입비중` 칸에 그대로 산출해 기입)**
```
C_lane     = 테마 lane 배정 자본 (예: 총자산의 15~20%, §4에서 캡)
risk_trade = 0.02 × C_lane            # 트레이드당 리스크 2%
ATR        = 최근 14일 일봉 ATR (원)
stop_dist  = k × ATR                  # k = 3.0 (KOSDAQ테마 seed, 넓게→수량 작게)
shares     = floor( risk_trade / stop_dist )
order_cap  = 0.01 × ADTV_원           # 주문은 일평균거래대금의 1% 이내
shares     = min(shares, order_cap / entry)
```
- **k=3.0 출발 이유**: 코스닥 테마는 ±30% 폭·VI로 일중 노이즈가 커서 k가 작으면 휩쏘로 털림. 넓은 손절 = 자동으로 작은 수량 → 보수적. 검증 후 k∈{2.0, 2.5, 3.0}로 조정.
- **ADTV 캡은 진우 규모에선 거의 안 걸림**(워치리스트 최소 ADTV도 약 230억) → 단, 1% 룰은 안전장치로 유지.

**하한가 안전여유 (필수, §아래 한국 특수성).**
```
stop_pct = stop_dist / entry
if stop_pct > 0.20:        # 손절폭이 −20% 초과면
    → 진입 스킵 또는 stop을 −20%로 고정하고 그에 맞춰 수량 재계산
```
이유: −30% 하한가에 걸리면 **손절 자체가 체결 불가**. 손절선은 −20% 이내에서 반드시 먼저 작동.

# 3. 테마 lane — 손절·청산 규율 (지금 적용)

**진입(분할).** 정찰 1/3 → 추세·거래량 확인 후 잔여 2/3. "한 방 전량" 금지.
**손절 (OR 조건, 먼저 닿는 것).**
- 가격: `entry − 3×ATR` (또는 §2의 −20% 캡).
- 구조: 전 저점 또는 20일 이동평균 종가 이탈.
- 갭 대비: 손절은 **전일 종가 기준**으로도 평가(시초 갭하락 대응).
**과열 회피 / 진입 보류.**
- 직전 봉 대비 +X% 급등(seed X=10%) 시 당일 신규진입 보류.
- **VI 발동·상한가 호가 시 신규진입 금지** (한국 특화 트리거 — VI를 보류 신호로 직접 사용).
**청산(분할).** 1차 목표 도달 시 1/2 익절 → 잔량은 ATR(3×) 트레일링으로 추세 종료까지. *트레일링은 추세 확인 시에만(검증 후 정식 채택).*
**무효화 트리거.** 워치리스트 `무효화_트리거` 칸에 thesis 붕괴 조건(예: 핵심 모멘텀 뉴스 부정/실적 미스)을 사전 기입 → 닿으면 손절 무관하게 청산.

# 4. 테마 lane — 집중·노출 한도 (지금 적용)

**근거.** 워치리스트가 바이오 3종목 → 동일 테마 동반 급락 리스크. 명목 분산 ≠ 실제 분산.
```
lane 총비중      ≤ 총자산 20%        # seed (본체 변동성 예산 안에서)
동일 테마        ≤ 2종목 AND ≤ lane의 40%
1종목 진입비중   ≤ lane의 25% (= 총자산 ≤ 5%)
동시 보유 종목수 ≤ 4~5
```
- 바이오가 현재 3종목이므로 **2종목으로 축소**하거나 비중을 40% 캡 안에 맞출 것.

**lane 손실 한도 (킬 스위치).**
```
월 누적 lane 손실 ≤ −15%  → 당월 신규진입 정지 (보유분은 손절 규율 유지)
분기 누적        ≤ −25%  → 전량 점검·축소
```
- seed 근거: 본체 최악월이 −9.79%인데, 고변동 테마 lane은 그 1.5배 수준(−15%)을 1차 차단선으로.

# 5. 본체 — 리밸런스 / MDD 메타룰 (검증 후, 우선순위 조정됨)

**밴드 리밸런스 — 하향 조정.** 자체 회전율 연구가 *진우 소액 계좌에선 월간이 비용 차감 후에도 최적*이라 입증 → **밴드의 비용절감 효과는 미미.** 따라서 "월간 유지"가 정답. 밴드는 *회전율 미세 절감* 목적의 선택적 옵션으로만(예: 목표 대비 상대 드리프트 ±20% 이내 종목은 그달 거래 스킵). 굳이 안 바꿔도 됨.

**MDD 메타룰 (지금 사전등록 권장 — 운영 안전장치).**
```
백테스트 MDD = −12.21%, 최악월 = −9.79%
[경보]  실현 월수익 < −10%   → 점검 플래그(역대 최악월 갱신)
[축소]  실현 낙폭 < −18%(≈백테스트 1.5×) → 노출 축소·전수 점검
```
- 이건 신호가 아니라 행동 규칙이라 과적합 위험 낮음 → 지금 룰화.
