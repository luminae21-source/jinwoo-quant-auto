## 0. 한 줄

**VQ(밸류+퀄리티)는 수익 엔진이 아니라 모멘텀 크래시 대비 꼬리 보험이다.** 평시엔 보험료(언더퍼폼)를 내고, 반도체/모멘텀이 무너질 때 값을 한다. 과열이 심할수록 보험 비중을 늘린다.

---

## 1. 등록 근거 (가설 → 한국 증거)

- **가설(원문):** QMJ(Asness·Frazzini·Pedersen) — "버블기엔 퀄리티가 싸지고(언더퍼폼), 낮은 quality 가격이 *높은 미래 수익을 예측*." 퀄리티 = distress기 flight-to-quality. (QARP = 밸류+퀄리티, Graham-Dodd 1934 계보.)
- **한국 실증(2026-06-13, 진우 데이터):**
  - 버블기(모멘텀强): VQ가 모멘텀에 **월 −4.5%p** 언더퍼폼 (보험료).
  - 모멘텀 최악 10개월(크래시): VQ **+9.6%p** 압도 (보험금).
  - 완만한 되돌림: −0.58%p(거의 동등). 상관(VQ−Mom vs 12M모멘텀) −0.30.
  - → **보험은 *꼬리(크래시)에 집중*** = 일상우위 아님. 정확히 "보험" 성격.
- (Asness ValMom: 밸류·모멘텀 결합 권고. 단 한국선 +0.57 양상관이라 *일상 분산*은 약하고 *꼬리 보험*만 유효.)

---

## 2. 운용 규칙

| 항목 | 규칙 |
|---|---|
| **역할** | 꼬리 보험. 수익 기대 금지. |
| **구성** | KOSPI 시총 top-200 중 VQ(밸류z+퀄리티z) 상위 15종목, 동일가중. `vq_insurance_sleeve.py`. |
| **비중** | base **15%** ~ max **30%**, 과열 연동. |
| **발동(과열 게이지)** | 반도체복합 평균 MA200 이격(%). 0~80%를 base~max로 선형. (gradecut RED와 같은 신호원.) |
| **증액** | RED(이격 高·피크 집중) = 보험 비중↑(크래시 위험 클 때 보험 더 산다). |
| **축소** | 크래시 통과·과열 정상화(GREEN) 후 base로 환원. |
| **재원** | 공격형(grade-cut)·반도체 트림분을 보험으로 (현금/방어와 함께). |

> 핵심: 보험 비중은 *수익률 보고* 늘리는 게 아니라 *과열(위험) 보고* 늘린다. 평시 언더퍼폼은 보험료로 받아들인다.

## 3. 현재 상태 (2026-06-13)

```
반도체복합 MA200 이격 68.9% → 신호 RED → 권장 보험 비중 27.9% (max 근접)
VQ-top 픽: 이마트·한국전력·한국가스공사·금호타이어·BGF리테일·롯데쇼핑·GS리테일·동원산업 등
           (저평가-퀄리티 방어주 = 밸류업 수혜 + flight-to-quality)
```

→ 지금은 반도체 과열이 심해 **보험을 최대치 가까이 권장**하는 국면. 픽이 전부 싼 퀄리티 방어주(유틸·소매·지주)라 반도체/모멘텀 되돌림 시 쿠션 기대.

## 4. 정직성·한계

- **보험료는 진짜다.** 버블이 더 가면 VQ는 계속 뒤진다(월 −4.5%p). 큰 하락을 견딜 수 있으면 보험 없이 가도 됨 — *선택*.
- 한국실증은 2021-26 단일구간·생존편향. 모멘텀 크래시 표본이 적음(최악 10개월). backtest ≠ 미래.
- VQ는 시장(EW/CW) 대비 음의 IR(평시) — *이건 의도된 것*(수익 아닌 보험).
- 집행·비중 최종 판단은 진우(눈·손·forward). 본 도구는 픽·비중 *제안*만.
