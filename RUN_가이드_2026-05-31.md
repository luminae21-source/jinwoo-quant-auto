## 2. Step 3 — 한국 FF/Carhart attribution
> ⚠️ `--market`·`--top-n`을 **두 명령에서 동일하게** (종목코드 정렬). DART 키는 `.dart_key`/`dart_config.json` 자동 사용.
```powershell
python fetch_dart_book_equity.py --market KOSPI --top-n 500 --start-year 2020 --end-year 2025
python build_korea_factors.py    --market KOSPI --top-n 500 --book-csv book_equity.csv --rf-annual 3.5
python run_attribution.py
```
`run_attribution.py` 출력:
- 팩터 적재(coef)·t값 + **alpha**(연환산·t값)
- t≥2면 "4팩터로 설명 안 되는 순수 alpha = 진우퀀트 고유 엣지", t<2면 엣지 재검토
- 최대 노출 팩터 = 그 팩터 부진 시 전략도 흔들리는 리스크 지점

(book 없이 빠르게 보려면 `fetch_dart_book_equity.py` 건너뛰고 `build_korea_factors.py`만 → MKT/SMB/WML만, HML 생략.)
