## 4. 옵션 B 전환 trigger (1-2개월 후)

### 자동 체크리스트

다음 4가지 **모두 True**일 때 옵션 B (ECHO_WEIGHT 1.2) 전환 검토:

- [ ] 1-2개월 실전 alpha vs 백테스트: **±2%p 이내**
  - 백테스트 예상 월수익: 약 +5.7%/월 (CAGR 71.54% / 12)
  - 실전 평균: -3.7% ~ +13.7% 범위 이내면 정상
- [ ] 큰 운영 문제 없음 (에러·지연·panic 매매 0건)
- [ ] 실전 MDD < **-12.5%** (백테스트 -12.21% + 여유)
- [ ] 진우님 시스템 운영 익숙해짐 (3개월 이상 매월 루틴 수행)

### 전환 절차

조건 충족 시:
```powershell

ECHO_WEIGHT = 1.0   →   ECHO_WEIGHT = 1.2
```
저장 후 다음 매월 1일 자동 적용.

**기대 효과**:
- CAGR +0.45%p (71.54% → 71.99%)
- Sharpe 거의 동일 (2.78 → 2.80)
- MDD 동일 (-12.21%)
- 위험 대비 reward 약간 증가

## 5. GitHub Actions 자동화 (선택)

현재 v3.6이 매일 16:30 KST 자동 실행 중. v3.7.2로 교체:

```yaml
# .github/workflows/jinwoo-quant-auto.yml
name: Jinwoo Quant Auto
on:
  schedule:
    - cron: '30 7 * * 1-5'  # 매일 16:30 KST (UTC 07:30)
  workflow_dispatch:

jobs:
  scoring:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with: { python-version: '3.12' }
      - run: pip install -q finance-datareader pandas numpy
      - name: Run v3.7.2 scoring
        run: python score_v37_2.py    # ← v3.6 자리에 교체
      - name: Deploy dashboard
        run: |
          cp dashboard_v37_2.html docs/index.html
          # GitHub Pages 자동 배포
```

자동화 후:
- 매일 16:30 KST 자동 점수 갱신
- 진우님은 매월 첫 영업일에 결과 확인 + 매매 결정
- 대시보드 URL에서 모바일 확인 가능
