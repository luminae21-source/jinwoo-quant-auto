## 1. 본체 v3.7.2 (실제 운용)

**유니버스·비중**
- **종목 수**: KOSPI **18종** 고정 풀에서 점수화. **동일가중**(equal weight = 고른 비중)이 출발.
- **종목 cap(상한) `STOCK_CAP = 0.15`** = 한 종목 최대 15%. *왜*: 한 종목 쏠림 방지(`score_v37_2.py`).
- **섹터 cap `SECTOR_CAP = 0.35`** = 한 산업 최대 35%. *왜*: 반도체가 33%까지 차서 한 업종 동반 급락 위험 완화(검증상 alpha 손실 −0.17%p로 거의 무해).
- 비중 계산: 1/n(고른 비중) → 종목 15% 컷 → 섹터 35% 컷 → 합 100%로 정규화.

**점수 = "체력_최종" (6팩터 + 보조 합산)**. 각 항목이 종목마다 점수를 더하거나 빼서 총점을 만든다.

| 항목 | 실제 산식·값 | 쉬운 뜻 | 큰 값/작은 값 의미 |
|---|---|---|---|
| **F_korean→체력_12점** | `F_korean × (12/9.001)` (F_korean=0~9 정수) | 한국형 재무체력(Piotroski 9점: 흑자·현금·부채↓ 등 9개 체크) 을 12점 만점으로 환산 | 높을수록 재무 튼튼 |
| **ModF** | 외부 입력 보정값(종목별) | 재무 수동 보정 | — |
| **FAR** | `r_1m<−5% & 체력_12점≥9 → +2`(FAR_BUY) / `r_1m>+5% & 체력≤4 → −2`(AVOID) / 약신호 ±1 | "튼튼한데 한 달새 급락"=역발상 매수 가점, "부실한데 급등"=회피 | +2=과대낙폭 우량주 |
| **Sloan** | 외부 입력(발생액 −2~+2) | 발생액(현금 안 들어온 장부이익)이 크면 이익의 질 낮음 | 음수=이익의 질 의심 |
| **Mom12** | `≥+60%→+2 / ≥+30%→+1 / ≥−10%→0 / ≥−30%→−1 / 그 미만→−2` | 최근 12개월 수익률(추세) | 클수록 상승추세 |
| **BAB** | `β<0.4→+2 / <0.7→+1 / <1.2→0 / <1.5→−1 / ≥1.5→−2` | 베타(β=시장 대비 출렁임). 저베타 우대 | β 작을수록 +(방어주) |
| **NOA** | 외부 입력(순영업자산 −2~+2) | 자산을 너무 키웠는지(과투자) 점검 | 음수=과투자 경계 |
| **Echo** | `t-12~t-7` 6개월 누적수익 3분위, 상위20% +1 / 하위20% −1, **×ECHO_WEIGHT=1.0** | "먼 과거(1년~7개월 전)" 모멘텀. 최근 6개월은 일부러 제외 | +1=먼과거 강세(반복 경향) |

- **체력_최종 = 체력_12점 + ModF + FAR + Sloan + Mom12 + BAB + NOA + Echo**
- **등급컷(실제, `grade()`)**: **S+ ≥ 14 / S ≥ 12 / A ≥ 9 / B ≥ 6 / C ≥ 3 / D ≥ 0 / F < 0**
- **매수 대상(picks)** = **S+·S·A** 등급만(`target_grades`). B 이하는 미편입.

**거래비용 `COST = 0.00235` = 0.235%**
- 뜻: 한 번 사고팔 때 드는 수수료+세금 **왕복 0.235%**. 백테스트가 이 비용을 차감하고도 이겨야 채택.

**MA200 추세방어**(채택된 유일한 시장 방어, `ma200_regime_check.py`)
- **`SMA_N = 10`** = 월간 10개월 이동평균(≈일봉 200일선). 종가가 이 선 **아래면 RISK-OFF**(관망), 위면 보유.
- *성격*: 신호가 아니라 **행동규칙**(월 1회 눈으로 점검). 휩쏘저감 확장은 검증서 기각 → 지금처럼 단순 유지.

## 2. v3.9 PEAD (병행 관찰 — 운용 미반영)

어닝 서프라이즈(실적이 예상보다 좋/나쁨)가 발표 후 한동안 주가에 천천히 반영되는 현상(PEAD)을 점수로.

| 항목 | 실제값 | 뜻 |
|---|---|---|
| **SUE** | `SUE = U_q / σ(직전 8개 U, ddof=1)`, 최소 6개 관측 | 표준화 어닝 서프라이즈 = 이번 분기 깜짝실적을 과거 변동폭으로 나눈 것 |
| **게이트** | `DRIFT_WINDOW_TD = 60` 거래일(캘린더 폴백 88일) | 실적공시 후 **60거래일 이내만** 점수 활성, 지나면 0 |
| **점수** | 18종 SUE 3분위, 상위20% +1 / 하위20% −1 | Echo와 동일 방식 |
| **가중치** | **`PEAD_WEIGHT = 0.5`** | 공식엔진 판정 ×0.5만 PASS(Δ+1.06%p net), ×1.0은 FAIL |
| 합산 | `체력_v39 = 체력_최종(v3.7.2) + 0.5 × SUE점수` | 본체 점수에 살짝(±0.5) 얹기 |

- **현재 = 기록만.** 운용 기준은 v3.7.2 그대로. **9월 초 판정**에서 채택 여부 결정.

## 3. 매매기법 도입안 (테마 lane — 집행·리스크)

**2%룰 + ATR 사이징**(`진우퀀트_손절사이징_룰_파라미터.md` §2)
```
risk_trade = 0.02 × C_lane          # 한 종목에 걸 위험 = lane 자본의 2%
stop_dist  = k × ATR(14)            # 손절폭, k=2.5(실데이터 확정), ATR=최근14일 일변동폭
shares     = floor(risk_trade / stop_dist)   # 손절폭이 넓으면 자동으로 수량↓
order_cap  = 0.01 × ADTV            # 주문은 일평균거래대금의 1% 이내
```
- *쉬운 뜻*: "틀려도 한 종목에서 lane 자본의 2%만 잃게" 수량을 거꾸로 계산. **손절선이 멀면 적게, 가까우면 많이.**

**실측 일ATR%**(`보강_트레이딩유형.md`, 월σ→일ATR 근사)

| 구분 | 월 변동성(σ) | 일ATR≈ | 뜻 |
|---|---|---|---|
| 본체(KOSPI) | 15.0% | **4.1%** | 하루 평균 ±4.1% 움직임 |
| KOSDAQ 전체 | 18.7% | **5.1%** | 더 출렁 |
| **워치 12종** | 26.7% | **7.3%** | 테마주가 가장 출렁(=수량 더 작게) |

**하한가 안전여유**: `stop_pct = stop_dist/entry > 0.20`이면 **진입 스킵 또는 −20%로 고정**. *왜*: −30% 하한가 걸리면 손절 자체가 체결 불가 → 손절선은 −20% 안에서 먼저 작동.

**킬스위치(lane NAV 기준, B6)**: **일 −7% / 주 −12% / 월 −20%** 도달 → **신규 진입 정지 + thesis 재검토**.
- *왜 이 값*: 워치 월σ 26.7%라 월 −20%는 약 0.75σ(정상범위 하단) — 그걸 넘으면 thesis 붕괴 신호. cap ≤10%라 전계좌 영향은 ×0.1.

**과열·선반영 회피(B3)**: 최근 **20일 +30%↑** 보류 / **52주 고점 −5% 이내** 보류 / 12-1 모멘텀 **+100%↑** late. → "카탈리스트가 아직 남았나" 재확인.

**집중·노출 한도**: lane 총비중 ≤ 전계좌 20%(위성 트랙은 ≤10%) · 동일 테마 ≤2종 & ≤lane 40% · 1종목 ≤lane 25% · 동시보유 ≤5.
