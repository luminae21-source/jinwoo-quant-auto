한 줄로. 엔진 = 퀄리티+모멘텀(검증됨). 방어 = MA200+퀄리티필터. 밸류·피크 = 매일 뜨는 경보(자동 트림 X — 사람이 판단). 지금 최우선 = 반도체 비중·트림 판단.
현실 기대 시장 +5~10%p(연 17~25%) · "76%"는 사후편향이라 안 씀.

1. 왜 이렇게 운영하나

PIT(그때 정보만으로) 백테스트 — 순수 팩터효과(IR):
슬리브IR판정
Q→Mom (퀄필터+모멘텀)+0.83엔진(최고)
Momentum+0.76엔진
Value 단독+0.08중립→선택도구
Quality 단독−0.21필터로만
① 알파는 종목 선별 + 집중 + 저회전에 있다. ② 밸류는 수익이 아니라 "비싼 과열주 거르기"에 쓴다. ③ 기계적 피크-트림은 백테스트 미달(비싼+과열=승자였음) → 경보로만.

3. 운영 루틴

매일 (자동) — 16:30 KST 자동 실행 → docs/overlay_scores.csv에 트림권고. 폰에서 TRIM 뜬 종목만 확인.
매월 (~10분)
1ma200_regime_check.py → 방어상태(RISK-OFF=추세 꺾임)
2holdings_trim_analysis.py → 보유 스냅샷
3TRIM 종목을 §4 규칙으로 판단 → 트림 결정
4(트림 시) trim_simulation.py로 영향 확인
분기 — value_quality_backtest.py로 모멘텀·Q→Mom 우위가 아직 유효한지 + DART 재무 갱신.

4. 의사결정 규칙 (사람이 판단)

경보 읽는 법
TRIM 비쌈+과열 → 차익실현 1순위 후보 · WATCH 비쌈, 과열 임박 → 관찰 · 순위Δ −2↓ = 강등됨
⚠️ 경보 = 자동매도 아님. "정상이익인가 피크이익인가?" 본인이 답한 뒤 결정.
트림하면 어디로? (검증 결과)
· 다른 주식 재투자 ✗ (조정장엔 다 같이 빠짐, 분산 안 됨)
· 현금/방어 ○ (Exit→현금: 변동성 27.7→19.0%, MDD −20.1→−16.6%)
· 트림폭 = 방어폭 — 반도체 −40% 시 Baseline −10.0% vs Trim½ −5.8% vs Exit −1.7%
밸류 기회 (역방향: 고퀄·저모멘텀)
점수 낮아도 사업 좋으면 기회. "2분 스토리" 못 쓰면 안 산다.
· 아모레퍼시픽 = 턴어라운드(북미 +35%, 바닥 탈출) → 밸류기회 쪽
· 삼양식품 = 초우량 고성장(ROE 30%)인데 비싸서 조정 → 트랩 아님, 합리적 가격 대기
