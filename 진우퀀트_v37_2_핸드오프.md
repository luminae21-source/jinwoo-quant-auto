
### Echo Momentum 정의 (Novy-Marx 2012)

```
Echo = (P_{t-7개월} / P_{t-12개월}) - 1
```

- t-12 ~ t-7 (중간 6개월) 누적수익률
- 최근 6개월 (t-6 ~ t-1)은 사용 안 함 (Novy-Marx 핵심 발견)
- 18종목 분위 점수:
  - 상위 20% (4종목) = +1
  - 중위 60% (10종목) = 0
  - 하위 20% (4종목) = -1
- 최종 score = Echo × 1.0

| GP+AG 결합 (MDD) | 노지혜 외 2023 결합 robust | MDD -9.79% (최저) | △ 위험 조절 보조 |

### 주요 학술적 발견

#### 1. F_korean·GP 강한 중복 (+0.644)
- Factor correlation matrix 분석 결과 (882 obs)
- Piotroski F-score 한국 적응 시 이미 quality 신호 포함
- 추가 GP factor는 incremental information 적음
- → **v3.6 F_korean이 한국에서 quality factor 역할 이미 수행 중**

#### 2. 단일 시점 quality cache의 lookahead bias
- 2025 사업보고서를 4년 OOS 전체에 적용 시 역방향 작동
- AG=-1 종목 (SK하이닉스·알테오젠·삼성물산)이 4년 강세 → 회피로 큰 손실
- AG=+1 종목 (KT&G·아모레·삼성SDI)이 4년 약세 → 픽으로 손실
- **교훈**: factor backtest는 PIT (Point-in-Time) 데이터 필수

#### 3. Cooper-Gulen-Schill 한국 적용 시 변형 작동
- 노지혜 외 2023 한국 25년에서는 robust이지만
- 18종목 대형주 universe에서는 PIT 적용 시도 -3.56%p (개선되나 부족)
- **MDD -9.79%는 최저** — 위험 조절 측면 가치 있음
- 18종목 universe는 사이클성 산업 (반도체·바이오·방산) 비중 높음 → AG 신호 역작용

#### 4. Echo Momentum의 한국 대형주 강력 작동
- Novy-Marx 2012 / 장지원 2017 한국 검증이 18종목에서도 작동
- 가격 데이터만 사용 → lookahead bias 없는 깨끗한 신호
- ECHO_WEIGHT 1.0이 0.5보다 우월 (+1.32%p → +3.65%p)
- 엄철준 2024 tail risk 우려 무관 (MDD 거의 동일)
