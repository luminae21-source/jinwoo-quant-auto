## 2. C-1 영역4 재정의 — Stage 0 사전등록 (Phase2, 승인 후 변경 금지)

### 2-1. 영역4의 위치 (게이트 성격 재정의)

영역4는 **'점수 개선'이 아니라 '보유/매도 판단 정밀화'** 레이어다. 따라서:
- 합격선은 **CAGR이 아니라 트리거 정확도·오분류 감소**로 설계한다. (시작prompt 원칙 + 영역3 §11 지시)
- v3.7.2 점수표(production)는 **무수정**. 영역4는 점수표가 S+로 잡고 있어도 "왜 빠졌나"를 분해해 매도 판단을 보조할 뿐, 점수에 끼어들지 않는다. (오늘 한미반도체: 점수표 S+(6위) vs attribution T1+T2 = 이 분리의 실증.)

### 2-2. Phase2 모델 (사전등록 ①)

```
r_i,t = α_i + β_M·r_KOSPI,t + β_S·r_sector(i),t + ε_i,t
```
- β는 종목별 시계열 회귀, **120영업일 rolling · 1일 lag** (look-ahead 차단). 60d는 노이즈 확인됨(실전로그 B: 삼성SDI 0.54→0.85 등 120d 수렴) → **120d 기본 채택**.
- sector(i) = 종목별 섹터 ETF/지수 (§2-4 매핑). 관측 가능한 ETF만 사용(18종 N 부족 → 자체 합성 factor 금지, Phase1 §5 통계한계 계승).
- 출력: market_contrib, sector_contrib, idio, idio_z. **분해 항등식 `r = market_contrib + sector_contrib + idio` 오차 0** 보존.

### 2-3. Phase2 합격선 (사전등록 ② — CAGR 아님, 변경 금지)

판정 표본 = **오늘(2026-06-07) 실데이터 T2 발동 11종**(attribution_v40_20260607_0904.json). Phase1 단일팩터가 T2로 잡은 종목에 2팩터를 재적용해 재분류한다.

T2 발동 11종 (cum_idio_20d %): 한미반도체 −46.6 / 두산에너빌리티 −38.5 / LIG넥스원 −35.9 / 한화에어로 −34.8 / ISC −33.8 / 삼성SDI −30.1 / NH투자증권 −29.6 / 아모레퍼시픽 −24.0 / 카카오 −21.7 / 삼양식품 −18.7 / 알테오젠 −14.9.
→ **한미반도체 1종 = 진성 종목약세**(1Q 어닝쇼크, 반도체 중 유일 역주행). **나머지 10종 = false-T2 후보.** 착시의 *원인*은 반도체 대형주(삼성전자·SK하이닉스 +20%대) 쏠림이 시총가중 KOSPI를 끌어올려 β_M·r_KOSPI(시장기여)를 과대평가 → 비반도체 종목의 idio가 음으로 과대. (10종이 반도체인 게 아니라 전 섹터에 걸쳐 있음.)

| 게이트 | 기준 | 통과 조건 |
|---|---|---|
| **A. 착시 재분류** | false-T2 후보 10종 (한미 제외) | 2팩터에서 sector_contrib가 약세를 흡수해 **≥30%(10종 중 ≥3종)가 T2 해제**(잔여 idio 누적 > −10%). ≥30%는 Phase1 §6 사전등록 prior 계승. 정확 재분류 수는 관측된 섹터 등락에 종속 → **결과 기록(게임 금지)**, 라인은 30%로 고정 |
| **B. 진성 약세 보존** | 한미반도체 | 2팩터에서도 **T2 유지**(반도체 섹터 빼고도 idio 누적 ≤ −10%). 거짓음성 금지 — 이게 더 중요 |
| **C. 분해 건전성** | 전 종목 매 영업일 | 분해 항등식 `r=mkt+sector+idio` 오차 0 (assert) + sector β 부호·크기 sanity (반도체주 β_S>0 유의) |

- **A·B·C 동시 충족 시 PASS** → Phase2 채택, 패널에 sector_contrib 칼럼 추가 + T2/T3 정밀화 반영.
- 하나라도 FAIL → Phase2 종결(섹터 팩터가 착시를 못 거르거나 진성 약세를 죽임), Phase1 단독 유지. **임계값(z·−10%·30%) 튜닝 금지.**
- 보조 기록: 재분류된 종목 각각의 sector_contrib 20일 누적을 병기 → 어느 섹터 동조였는지 사후 해석용(판정엔 미사용).
