# 진우퀀트 — 풀 시스템 PIT universe 검증 설계

작성: 2026-06-02. 목적: 18종목 *고정·재량선택*을 제거하고, **계산 가능·look-ahead 없는(PIT) 점수 시스템**을
광범위 KOSPI universe에 매월/매년 적용해 종목을 *그때 정보만으로* 뽑았을 때도 alpha가 살아남는지 검증.

## 0. 왜 이 테스트가 결정적인가
- PBO·DSR·attribution 은 모두 "18종목 고정"을 전제 → universe 선택 편향을 못 본다.
- `진우퀀트_v36_universe.md`가 자인: 5종목 교체가 **4년 백테스트 성과 기반**, LIG넥스원은 "등급 F인데 +513%라 등급 무시 유지". → universe에 명백한 hindsight.
- 따라서 **유일하게 남은 결정적 질문**: 재량 선택을 빼고 *체계적·PIT*로 돌려도 엣지가 남는가?

## 1. 핵심 한계 — F_korean은 proxy할 수밖에 없음 (정직)
- 점수: `체력_최종 = F_korean×1.33 + ModF + FAR + Sloan + Mom12 + BAB + NOA`
- **F_korean(품질점수 0~9)이 지배 항이자 핵심 알파원**(α검증: F_korean L-S +20.89%)인데, **코드에 산식이 없고 종목별 수기 부여**값.
- ModF(산업보정 F)·Sloan(accrual)·NOA 도 일부 수기.
- → 정확 재현 불가 ⇒ **계산 가능한 표준 대용치로 치환**:
  | 원 점수 | PIT 대용(계산식) | 데이터 |
  |---|---|---|
  | F_korean | **Piotroski F-score (0~9)** | DART 재무 9개 기준 |
  | ModF | 산업 내 Piotroski 상대순위 | DART |
  | Sloan | accrual = (NI − CFO)/총자산 | DART |
  | NOA | (총자산−현금−총부채+차입금)/직전총자산, 산업 z | DART |
  | FAR·Mom12·BAB·Echo | 가격 PIT (backtest_v37_2에 이미 구현) | FDR |
- **해석 규약(가장 중요)**:
  - PIT-systematic이 **alpha 유지** → 엣지가 *체계적·반복가능*. 진우퀀트의 가치 입증.
  - PIT-systematic이 **시장 수준으로 붕괴** → 진짜 alpha는 **F_korean 재량 부여(=analyst 판단/hindsight)**에 있었음 → 라이브 반복성 의문.
  - 즉 이 테스트는 "진우퀀트 정확 시스템"이 아니라 **"그 시스템의 정직한 계산가능 근사판"**을 검증한다(이 한계를 결과에 반드시 명시).

## 2. Piotroski F-score (F_korean 대용, 9점)
회계연도 t vs t-1 (DART 사업보고서 11011, PIT: 5월 이후 직전 회계연도 사용 — 공시 시차):
- 수익성(4): ROA>0 · CFO>0 · ΔROA>0 · CFO>NI(accrual 양호)
- 안정성(3): ΔLT부채비율<0 · ΔCurrent ratio>0 · 신주발행 없음(Δ주식수≤0)
- 효율(2): ΔGross margin>0 · ΔAsset turnover>0
→ 합 0~9 → 체력_12점 대용 = F × (12/9).

## 5. 정직한 caveats
- F_korean을 Piotroski로 **근사**(정확 재현 아님) → "근사판" 검증임을 결과에 명시.
- PIT 시총: 과거 시점 정밀 시총 없으면 현재 top-N 근사 → 약한 survivorship 잔존(완화: 넉넉한 top-N, 상장폐지 종목 포함 권장).
- DART 분기/연 공시 시차(사업보고서 ~3월) → 5월 이후 직전연도 사용으로 룩어헤드 차단.
- T≈49개월·단일 시장 → 결과는 방향성.
