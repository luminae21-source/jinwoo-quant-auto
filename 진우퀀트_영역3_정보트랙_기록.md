## 1. 선정 룰 (파일·상수 고정 — 변경 시 이 표부터 갱신)

**풀**: 합본 가격패널 `kospi_monthly_prices.csv` (584종목) → 단 유동성 가드레일이 `liquidity_sector.csv`(KOSPI 전용) 기준이라 유효 풀 = **KOSPI**. (KOSDAQ는 별도 관찰 트랙)

**랭킹**: PIT 점수 `pit_universe_backtest.score_at()` = F×(12/9) + Sloan + NOA + Mom12 + BAB(β) + Echo. (ModF·FAR 제외 = production 체력과 다른 경량 proxy)

**필터 상수** (`universe_rules.py`):

| 상수 | 값 | 의미 |
|---|---|---|
| `PIOTROSKI_MIN` | 6 | F ≥ 6 |
| `_latest_two()` | — | 재무 비악화 (ROA −2%p↓ 또는 적자/CFO 음수 → 제외) |
| `MIN_MONTHS` | 12 | 가격이력 ≥ 12개월 |
| `SIZE_TOP` | 200 | 시총 상위 200 |
| `LIQ_KEEP_PCT` | 0.60 | ADTV 상위 60% (절대 거래대금 하한 아님) |
| `SECTOR_CAP` | 3 | KRX 세분류 섹터당 ≤3 |
| `MAX_SWAP_Q` | 2 | 분기당 교체 ≤2 |

**종목 수 top-30 근거**: `sweep_universe_size.csv` — 18/30/50/100 중 30이 Sharpe 1.00·IR 0.12로 최선.
**carve-in 대신 pooled 근거**: MR_28+2·26+4 둘 다 사전 게이트(IR ≥ pooled−0.02) FAIL (`universe_mr_validate_20260605_1219.json`).
