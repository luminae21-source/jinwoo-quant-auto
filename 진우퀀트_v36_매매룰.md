## 매매 4-Layer

```
1. Selection (등급) → 매매 후보
2. Timing (FAR + 시장) → 진입·청산 시점
3. Sizing (Kelly) → 비중 결정
4. Exit (트레일링 스톱) → 손절·익절
```

|---|---|---|---|

## 2. FAR 신호 매매

### FAR_BUY (1M -5%↓ + 체력 9+)

```
✦ 4분할 매수 (1주 간격, 25%씩)
✦ 총 비중: 등급별 기본 × 1.5배 허용
✦ 시장점수 ≥ 12 일 때만 발동
✦ 시장점수 < 12 → 비중 1.0배만
```

**역사적 알파:** 2023 +28.45%, 2025 -16.45% (강세장 약점)

### FAR_AVOID (1M +5%↑ + 체력 4↓)

```
✦ 발동 즉시 -50% 비중 축소
✦ 다음 분기 점수 재확인
✦ 추가 악화 시 잔여 청산
```

**역사적 알파:** -7.05% (일관 입증)

## 3. Kelly 사이징 (체력 비례)

```python
비중 = max(2%, min(10%, 체력_최종 × 0.5%))
```

| 체력 | 비중 |
|---|---|
| 14+ (S+) | 7~10% |
| 12~14 (S) | 6~7% |
| 9~12 (A) | 5% |
| 6~9 (B) | 3.5% |
| 3~6 (C) | 2% (최소) |
| 0~3 (D) | 2% (최소) |
| <0 (F) | 0% or 청산 |

**총 비중 ≤ 95%** (현금 5%+ 유지)

## 8. 핵심 원칙 (잊지 말 것)

1. **점수는 *예측*이지 *결과*가 아님** — 분기 점수 변화 추적이 핵심
2. **약세장에서 진우퀀트가 진가** — 시장점수 < 12 시 보수적
3. **단일 종목 집착 X** — 분산 의무 절대 준수
4. **FAR_BUY는 강세장에서 X** — 시장점수 필터 적용
5. **매도는 점수 따라, 매수는 점수 + 타이밍** — 비대칭 룰
