→ **통합-0 미충족(데이터 2/4).** 블루프린트 §0이 명시: *"지금 통합하면 미완성 상태를 고정."* **통합 빌드는 보류**가 맞다.

### 지금 할 수 있는 통합 준비 (빌드 아닌 설계 — 데이터 불필요)
**`data.json` 스키마 사전 설계** = 측정이 쌓이면 바로 꽂도록 그릇만 미리 확정. 엔진 무수정, 어댑터가 읽어 채울 한 개 파일:
```
{
  "asof": "YYYY-MM-DD",
  "market": { "kospi":{"ma200":"위/아래","mtd":-0.07}, "kosdaq":{...}, "vkospi":"보통",
              "calendar":[{"d":"7/16","e":"금통위"}] },
  "system": { "rebalDue":"7/1", "picks":[{"name":"한화에어로","grade":"S+","w":7.69}, ...],
              "ma200":"위" },
  "theme":  { "watch":[{"name":"펩트론","gate":"PASS","hold":"H-C","invalidation":"릴리 무산"}, ...] },
  "trackw": { "months":[{"m":"2026-06","W":null,"sys":null,"themeEW":null}] },
  "dict":   "용어사전 링크" }
```
- 5탭(오늘·시스템·테마·기록·사전)이 각각 위 키 하나씩 읽음. **허브는 계산 0 — data.json만 그림**(블루프린트 §4).
- 이 스키마만 확정해두면, 6월말 첫 측정 후 어댑터 1개 + 단일 HTML로 **통합-0 즉시 착수 가능**.
