### Echo Momentum 정의 (Claude 초기 오류 수정)
- **Echo (Novy-Marx 2012)**: t-12 ~ t-7 (중간기간) 누적수익률
- **6M-12M 가속**: 6M return − 12M return ← Echo와 **다른 신호**
- 두 신호는 일부 케이스 반대 부호
- **v3.8에서는 Echo 우선 검증**

### 운영 측면 별도 검증 (GPT 지적)
- 거래비용 (한국 매매수수료 + 세금 0.18%)
- 세후수익 (직장인 종합과세)
- 리밸런싱 안정성 (turnover, slippage)

### 데이터 소스
- FDR (가격)
- DART API (분기 재무: 자산총계, 매출원가, 영업이익)

### 5요소 통합 regime detector

| # | 요소 | 측정 | 데이터 |
|---|---|---|---|
| 1 | **지수 추세** | KOSPI 60/120/200일 MA | FDR |
| 2 | **VKOSPI** | 비대칭 룰 (이정환 외 2023) | KRX |
| 3 | **수급** | 외국인·기관 순매수 누적 | KRX |
| 4 | **섹터 집중도** | HHI 또는 top 3 섹터 비중 | 자체 계산 |
| 5 | **팩터 ON/OFF** | F-score·GP·BAB·Mom 최근 alpha 부호 | 자체 백테스트 |

### 한국 시장 학술 합의
- Cooper-Gutierrez-Hameed 2004 Up/Down state framework은 **한국에서 직접 적용 어려움**
- 한국은 음의 모멘텀이 일관 (엄철준 외 2020, 박종원 시리즈)
- 따라서 Faber/Whaley/Cooper 그대로 복사 X, 한국식으로 재설계
