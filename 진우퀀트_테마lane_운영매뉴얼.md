# 진우퀀트 — KOSDAQ 테마 lane 운영 매뉴얼 (통합)

> 2026-06-07 | 테마 lane 전 자산을 **단일 운영 문서**로 통합(방법론·스캔·대시보드·매매기법 명세·runbook). 각 단계에 구체 임계 박음.
> production·C·D·영역3·v41 **무수정**. **자동매매 아님 — 종목·thesis·매수·집행은 진우.** 기존 `진우퀀트_KOSDAQ_월간루틴_runbook.md`는 이 매뉴얼로 통합.

## 0. 한 장 요약 (냉장고 카드)
- **backward = 시스템**(KOSPI v3.7.2, 손대지 말 것) / **forward = 진우**(KOSDAQ 테마)
- 후보 승격 = **신호 2+ 점등 AND 선반영 통과 AND 가드레일 PASS AND thesis·무효화 등록**
- 사이즈 = **2%룰·3ATR → 테마 ≤9% · 슬롯 ≤10% · 동일테마 ≤6%**
- 손절 = **3ATR(≈22%, 하한가 −30% 전 작동)** · 킬스위치 = **일 −7 / 주 −12 / 월 −20%**
- 매달 **Track W 기록**, **6개월 누적 반사실 ≥ 0**이면 유지

## 2. per-trade 체크리스트 (사기 전 7개 — 전부 ✅여야 진입)

1. **thesis 1줄** 있나? (왜 *지금* = 카탈리스트 1개)
2. **신호 2+ 점등?** (A 촉매 / B 확산 / C 수급·어텐션 중 둘 이상, 검색량 단독 아님)
3. **선반영 통과?** (최근 20일 < +30% AND 52주 고점 −5% 밖)
4. **가드레일 PASS?** (`kosdaq_theme_guardrail.py`)
5. **무효화 트리거 사전등록?** ("이러면 손절/철회" — 사기 전에)
6. **사이즈 ≤ min(2%룰 9%, 슬롯 10%)** AND 동일테마 ≤6%?
7. **손절선(진입가 −3ATR) 적어놨나?** (하한가 전인가)

→ **하나라도 ✗면 보류.** 가드레일 통과 = '고려 가능'일 뿐 매수신호 아님. 매수는 진우 직접.

- 매달 **Track W 반사실** = 픽 수익 − 같은 돈 시스템 픽 (보조: 테마 EW · KOSDAQ150 · MKT)
- **중간점검 2026-09**, **본판정 2026-12말**: 6개월 누적 반사실 **≥ 0 → 유지/확대**, **음수 지속 → 축소·중단**
- 무효화 트리거 닿으면 **즉시 정리**, 분기 thesis 재심사

## 5. 지금 적용 가능 vs 검증 후

| | 항목 |
|---|---|
| **지금 적용 가능**(룰·집행·사이징) | 스캔·선반영 보류·가드레일·thesis/무효화·2%룰 사이징·동일테마 한도·킬스위치·VWAP 모니터(±10/20bp)·Track W 기록 |
| **검증 후 채택**(forward 측정) | 분할 횟수·간격(B2)·ATR 손절 k값(B5) — 테마는 백테스트 불가 → Track W로 효과 측정 |
| **금지** | 본체(KOSPI) 신규 오버레이(이미 백테스트 기각, VWAP·MA200만 유지) |

> 정직 가드: 백테스트 ≠ 미래 · 발굴·점등 = 후보 surfacing(매수신호 아님) · hindsight 금지. **본체는 손대지 말 것.** 검증 = 테마 forward(Track W).
