## 4. 설계 3원칙 (진우 방향 — 명문화)

- **가. 선정기준 = 성장성 + 코스닥 회계특성 재무적합성.** KOSDAQ는 성장주 universe. 가치/대형주 quality 직이식 금지. **성장(매출·이익·CFO 모멘텀)을 1차 축**, 재무건전성은 *코스닥 회계 변동성에 적합성 검증된 팩터만* 채택.
- **나. 팩터 상관 = 중복신호 prune 용도(적재 X).** 상관 높은 지표를 둘 다 싣지 않는다. Stage 2에서 상관행렬 먼저 → |ρ| 높은 쌍은 하나만. 근거 = v3.8 F_korean·GP **+0.644** 중복이 alpha 손실 원인. 상관은 "추가"가 아니라 **"쳐내는"** 도구.
- **다. 합격선 먼저 등록 → 그 다음에만 fitting.** 결과 보고 합격선·팩터셋 변경 금지(누적 기각 9건의 규율 그대로).

## 5. 후보 지표 설계 방향 (Stage 1~2 입력 — 확정 아님, prune 전 후보군)

**축1 — 성장성(1차 축, `fundamentals_kosdaq.csv` 연간 YoY로 산출 가능)**
매출 YoY · 영업이익 YoY · 순이익 YoY · CFO YoY(현금동반 성장) · 다년 성장 안정성/추세.
⚠️ 적자기업 多 → 이익성장률 0기준·부호반전 왜곡 → **매출·CFO를 robust 축**, 이익은 흑자전환/적자지속 더미 보조.

**축2 — 코스닥 회계적합 quality (KOSPI 가중치 직이식 금지, 부호 재검증 후 채택)**
한상욱 F_korean 부호 코스닥 재검증(F4 Accrual·F5 ΔLever 한국 페널티 유지되나?) · Sloan accrual(cfo−net_income 괴리; 코스닥 회계변동성↑라 더 중요할 수 있음) · 수익성 수준(ROA·op margin: "수익 있는 성장" 선별) · 재무건전성(부채·유동비율: 소형주 부도 가드).

**축3 — 상관 prune (적재 아님)**
Stage 2에서 후보 전체 상관행렬(Pearson·Spearman) → |ρ| 높은 쌍 하나만. `kosdaq_factors.csv`(MKT/SMB/WML/RF; **HML 결측**)로 코스닥 자체 팩터 부호/IC·WML(모멘텀)과 성장의 상관 점검(한국 F–모멘텀 강상관 전례). 산출물 = **채택 팩터셋 + 적재금지 목록**.

**밸류·사이즈 주의**: size·value 약효 + HML 결측 → value 1차축 배제, 성장 중심이 정합.

## 6. 사전 합격선 (Go/No-Go) — **높게** [✅ 진우 승인·동결 2026-06-06]

**왜 높게**: ① 영역3 확장이 **−3.46%p로 기각**(게이트 A: ≥+3%p·IR≥0.30·MDD비악화). ② KOSPI-only 실측에서 KOSDAQ 룰 체계적 알파 ≈0(시장초과 음수). **이 모듈은 음의 prior에서 출발 → 어설픈 +는 noise.**

### 6-1. Base 비교 대상 (D 교훈: 절대값 참조 금지 → 동시점 병행 delta 판정)
- **판정 base = 동시점 공식 실행으로 산출한 "KOSDAQ 시장 수익".** 절대 CAGR 참조 금지(2026-06 급락이 base 흔든 전례).
- 시계열 후보(폴더 실측): **(a) `kosdaq_factors.csv` MKT**(2021-01~, 시장수익 프록시) · **(b) 가격패널 등가중**(2019-12~) · **(c) KOSDAQ150 공식지수(파일 없음 → PC fetch 필요)**. → **§8 결정 1.**

### 6-2. 측정 축 — 3중 동시 충족(AND), 코스닥 비용 0.5~0.7% 차감 후 net
1. **시장초과 알파**: 연환산 시장초과(vs base) **≥ +3.0%p** (영역3 게이트 A와 동일 눈높이, +1%p보다 훨씬 높게).
2. **위험조정**: **IR ≥ +0.30** AND Sharpe 벤치 비열위(−0.01 허용).
3. **드로다운**: MDD 벤치 대비 **비악화**(악화 ≤ 0%p). 음의 prior라 영역3의 "+2%p 허용"보다 타이트하게.

### 6-3. 표본·변형·처리
- **다년 OOS PIT**(거래일 정렬·공시지연·look-ahead 없음). 윈도우 = 가격∩팩터 교집합(실질 2021~ 또는 가능 최장).
- **변형 2개만** 사전 등록(그리드 금지) — 예: ⓐ 팩터셋 A/B **또는** ⓑ 성장가중 ×0.5/×1.0. → **§8 결정 2.**
- **FAIL → 즉시 종료**, 10번째 사전등록 기각 기록, 재튜닝·그리드 금지.

### 6-4. 코스닥 특수 게이트 (선행 조건)
- **유효 N 게이트**: 매월 채점 가능 종목 ≥ (제안: **40종**, 재무 가용 297·유동성 필터 후 분산 확보선). 미달 시 Stage 1에서 종료.
