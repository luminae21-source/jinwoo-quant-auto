## 2. v3.7 핵심 변화

### 점수 공식
```
v3.6: 체력_최종 = 체력_12점 + ModF + FAR + Sloan
v3.7: 체력_최종 = 체력_12점 + ModF + FAR + Sloan + Mom12 + BAB + NOA
                                                     └ +6 ~ -6 추가 ┘
```

### 신규 3팩터
| 팩터 | 데이터 | 갱신 | 시그널 |
|---|---|---|---|
| **Mom12** (12M Momentum) | FDR 12M 일별 종가 | 매일 | LIG넥스원·NAVER 케이스 보강 |
| **BAB** (β_60d) | FDR 60일 + KOSPI | 매일 | 고변동 종목 risk-adjusted 페널티 |
| **NOA** (자산효율) | DART 분기 재무 | **분기 1회 수동** | 회계 품질 (Sloan 보완) |

### 회귀 안전성
- `USE_V37_FACTORS = False` 시 v3.6 동작 (Mom12·BAB·NOA = 0)
- `JINWOO_v37` dict의 NOA 키 초기값은 모두 0 → DART 분기 작업 전엔 NOA 영향 없음
- 즉 첫 실행 시 자동으로 작동하는 건 Mom12·BAB 2개. NOA는 단계적 도입.

## 4. 알려진 한계 (전부 명세 문서에 상술)

### 백테스트 한계
- F·ModF·Sloan·NOA는 **현재(2026-05) 값을 정적 사용** → look-ahead bias 있음
- 진정한 walk-forward는 영역 4 attribution 단계에서 시점별 F-score 재계산 시 가능
- **본 백테스트는 "현재 시스템을 4년 전부터 운용했다면" 시뮬레이션** — 한계 인지 필요

### 코드 한계
- sandbox 환경에서 FDR 실제 호출 불가 (인터넷 차단) → 진우님 PC 실행 전제
- NOA 산업 z-score는 18종목·11산업 분포에서 N=1~2 산업 다수 → 초기엔 수동 평가 후 입력 권장

### 임계값 의존성
- Mom12 ±60% / ±30%, BAB 0.7 / 0.9 / 1.1 / 1.3 — 모두 학술 표준 + 한국 시장 평균치 기반
- **백테스트 결과 따라 미세조정 가능**하지만 overfitting 위험 → 자제 권장 (단순함 우선)

## 5. 영역 2 (매매 기법)로 넘어갈 때 — 영역 1의 시사점

영역 1에서 발견한 사실들 중 영역 2가 활용해야 할 것:

### 5-1. BAB는 자연스러운 sizing 신호
BAB가 +2인 저β 종목은 Kelly 비중 상한 ↑, BAB −2 고β는 비중 상한 ↓.
v3.6 매매룰의 `비중 = max(2%, min(10%, 체력 × 0.5%))`에 BAB 가산항 검토:
```
비중 = max(2%, min(10%, (체력 + 0.5×BAB) × 0.5%))
```

### 5-2. Mom12는 진입 타이밍에 활용
Mom12 +2 (강한 추세) → 4분할 매수 간격 짧게 (1주 내 완료)
Mom12 0 또는 음수 → 4분할 매수 간격 길게 (1~2개월 분산)

### 5-3. NOA는 장기 보유 종목 선별
NOA +2 (효율적)는 장기 핵심 보유, NOA −2는 단기 모멘텀 트레이딩만 (자산 비효율은 모멘텀이 끊기면 약점 부각).

### 5-4. 영역 1이 영역 2에 넘기는 질문
- 신규 3팩터 점수가 매매룰의 어느 layer에 들어가야 하나? (Selection? Sizing? Exit?)
- Mom12 vs FAR의 충돌 케이스 (Mom12 −2 + FAR_BUY)는 어떻게 처리?
- BAB 기반 헤지 (BAB −2 종목 보유 시 KODEX 인버스 일부 페어)?
